# Wedding RSVP

使用 `Next.js`、`Tailwind CSS`、`react-hook-form`、`zod` 與 `Firebase Firestore` 建立的婚禮邀請回覆網站。

## 功能

- 首頁展示婚禮資訊
- RSVP 表單驗證與送出
- Firebase Firestore 儲存賓客資料
- 新人後台查看回覆資料
- 座位安排 Tab 預留空白區

## 開始使用

1. 複製環境變數：

```bash
cp .env.example .env.local
```

2. 填入 Firebase 專案設定與後台密碼
3. 啟動開發環境：

```bash
npm run dev
```
